## How to Use This Manual

This manual is organized in the order you would typically set up and use the GenkÅ platform:

1. **First-time users** should start with [Getting Started](./01-getting-started.md) and work through the setup sections.
2. **Daily operations** are covered in the Patient Management, Scheduling, and Communications sections.
3. **Administrative tasks** are covered in Staff Management, Analytics, and Settings.

Each section includes:
- Step-by-step instructions with screenshots placeholders
- Best practices and tips
- Common issues and solutions

## Navigation Tips

- Use the sidebar navigation in the admin portal to access different sections
- The global search bar (Ctrl/Cmd + K) allows quick access to patients, appointments, and settings
- Breadcrumbs at the top of each page show your current location
