このクイックリファレンスガイドは、Genkōでの一般的なタスクとナビゲーションに関する一目でわかる情報を提供します。

## 一般的なタスクのクイックステップ

### 新規患者を追加

1. サイドバー → **患者**
2. **+ 患者を追加**をクリック
3. 必須フィールドを入力（名、姓、生年月日）
4. **保存**をクリック

### 予約をスケジュール

1. サイドバー → **スケジュール**
2. 空き時間帯をクリック または **+ 新規予約**
3. 患者とプロバイダーを選択
4. 日付、時間、所要時間を設定
5. **作成**をクリック

### スタッフメンバーを追加

1. サイドバー → **スタッフ**
2. **+ スタッフを追加**をクリック
3. 名前、メール、役割を入力
4. **保存**をクリック（招待が自動送信）

### 遠隔医療セッションを開始

1. サイドバー → **スケジュール** → **遠隔医療**タブ
2. スケジュールされたセッションを見つける
3. **セッションを開始**をクリック（予約時間近く）

### メッセージを送信

1. サイドバー → **メッセージ**
2. 会話を選択 または **新しい会話**をクリック
3. メッセージを入力
4. **送信**をクリック

### リードを患者に変換

1. サイドバー → **リード**
2. リードを見つける
3. ✅（変換）アイコンをクリック
4. 変換を確認

## 日次ワークフロー

管理者向け推奨日次ルーティン：

**朝：**
1. ダッシュボードで今日の概要を確認
2. アラートや通知を確認
3. 当日の予約を確認

**日中：**
4. 必要に応じて予約を管理
5. メッセージに応答
6. 新しいリードを処理

**終業時：**
7. 完了した予約を確認
8. 不参加を適切にマーク
9. 明日のスケジュールをプレビュー

## 緊急手順

### 患者がポータルにアクセスできない
1. 患者のメールが正しいか確認
2. パスワードリセットリンクを送信
3. 患者ステータスがアクティブか確認

### スタッフがログインできない
1. アカウントがアクティブか確認
2. パスワードリセットを送信
3. アカウントロックアウトを確認
4. 必要に応じて招待を再送信

### 支払い失敗
1. 支払い方法を確認
2. カード詳細を更新
3. 拒否された場合は銀行に連絡
4. 代替支払い方法を追加

### データが見つからない
1. 日付範囲フィルターを確認
2. 正しい組織か確認（複数ある場合）
3. 同期を待つ（最大15分）
4. 続く場合はサポートに連絡
