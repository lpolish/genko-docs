# ビジネスプロファイル管理

ビジネスプロファイルは、Genkō内でのクリニックの公開アイデンティティです。完全なプロファイルは患者との信頼を築き、スムーズな運営を確保します。

---

## ビジネスプロファイルへのアクセス

1. サイドバーの**ビジネスプロファイル**に移動（管理下）
2. またはダッシュボードのプロファイル完成バナーをクリック
3. URL：`/admin/business-profile`

---

## ビジネスプロファイルが重要な理由

完全なビジネスプロファイル：
- **患者の信頼を築く**：患者はクリニックについての専門的な情報を見る
- **機能を有効にする**：一部の機能には組織の詳細が必要
- **コミュニケーションを改善**：クリニック情報が患者通知に表示される
- **公開リストをサポート**：クリニックが公開ディレクトリに表示される可能性

---

## ビジネスプロファイルセクション

### 1. 組織情報

#### 基本詳細
| フィールド | 説明 | 必須 |
|------------|------|------|
| **組織名** | クリニックまたは診療所の名前 | はい |
| **法人名** | 公式登録ビジネス名 | いいえ |
| **説明** | 診療の簡単な説明 | 推奨 |

#### 連絡先情報
| フィールド | 説明 | フォーマット |
|------------|------|------------|
| **メール** | 主要連絡先メール | email@example.com |
| **電話** | 主要電話番号 | +81 (3) 1234-5678 |
| **ウェブサイト** | 診療所のウェブサイト | https://example.com |

#### 住所情報
| フィールド | 説明 |
|------------|------|
| **住所** | 建物番号と通り名 |
| **部屋/ユニット** | 追加住所行（オプション） |
| **市区町村** | 市区町村名 |
| **都道府県** | 都道府県 |
| **郵便番号** | 郵便番号 |
| **国** | 国（自動検出） |

### 2. ブランディングとロゴ

#### ロゴアップロード
- **対応フォーマット**：PNG、JPG、WEBP
- **推奨サイズ**：最小200x200ピクセル
- **最大ファイルサイズ**：2MB
- **形状**：正方形のロゴが最適

**ロゴをアップロードするには：**
1. ロゴアップロードエリアをクリックするか、画像をドラッグ＆ドロップ
2. 保存前にロゴをプレビュー
3. **保存**をクリックして変更を適用

#### ブランドカラー（近日公開）
- プライマリカラーの選択
- セカンダリカラーの選択
- アクセントカラーの選択

| 金曜日 | 09:00 | 17:00 | ☐ |

### 4. 診療専門分野

診療所が提供する専門分野を選択：
- 一般診療
- 家庭医学
- 小児科
- 内科
- 皮膚科
- 心臓科
- その他...

**専門分野を追加：**
1. **専門分野を追加**をクリック
2. ドロップダウンリストから選択
3. 追加の専門分野について繰り返し
4. **保存**をクリック

## プロファイル完成インジケーター

プロファイル完成インジケーターは進捗を表示：

| 完成度 | ステータス | アクション |
|--------|----------|----------|
| 0-25% | 🔴 未完成 | 必須フィールドを完成 |
| 26-50% | 🟡 基本 | 詳細を追加 |
| 51-75% | 🟡 良好 | ほぼ完成 |
| 76-99% | 🟢 もう少し | 最終調整を追加 |
| 100% | 🟢 完成 | プロファイル完全設定 |

**100%に必要：**
- 組織名
- メールアドレス
- 電話番号
- 住所
- ロゴ

## 変更の保存

変更は2つの方法で保存されます：

### 自動保存
- 一部のフィールドは入力時に自動保存
- 「保存中...」インジケーターを確認

### 手動保存
- 右上の**保存**ボタンをクリック
- 「正常に保存されました」の確認を待つ

**未保存の変更アラート：**
- 未保存の変更がある状態で離れようとすると、警告が表示
- 変更を保存または破棄を選択

## ベストプラクティス

### ロゴガイドライン
1. 最高品質のために高解像度画像を使用
2. 小さいサイズでもロゴが読み取れることを確認
3. シンプルでクリーンなデザインを使用
4. すべての素材で一貫したブランディングを維持

### 説明のヒント
1. 簡潔に保つ（2-3文）
2. 主要な専門分野に言及
3. ユニークな価値提案を含める
4. 専門的な言葉を使用

### 連絡先情報
1. 専用のビジネスメールを使用（個人用ではなく）
2. 営業時間中に電話が応答されることを確認
3. ウェブサイトURLを最新に保つ
4. 患者ナビゲーションのために住所が正確であることを確認

## プライバシーの考慮事項

ビジネスプロファイル情報は以下に表示される可能性：
- スタッフメンバー
- 患者
- 公開ディレクトリ（有効な場合）

**機密情報：**
- 財務詳細は**決して**公開表示されない
- スタッフの個人情報は保護される
- 患者数はプライベート

---

## トラブルシューティング

### ロゴがアップロードできない
- ファイル形式を確認（PNG、JPG、WEBPのみ）
- 2MBを超える場合はファイルサイズを縮小
- 別のブラウザを試す
- ブラウザキャッシュをクリア

### 変更が保存されない
- インターネット接続を確認
- エラーメッセージを探す
- ページを更新して再試行
- 問題が続く場合はサポートに連絡

### 住所検証エラー
- 住所形式が国に一致することを確認
- タイプミスを確認
- 住所コンポーネントを個別に入力してみる
