# 設定と構成

設定セクションでは、Genkōを診療所のニーズに合わせてカスタマイズできます。このガイドでは、管理者が利用できるすべての設定オプションをカバーします。

---

## 設定へのアクセス

1. サイドバーの**設定**をクリック
2. URL：`/admin/settings`

---

## 設定構造

設定はタブで整理されています：

| タブ | 目的 |
|------|------|
| **アカウント** | 個人アカウント設定 |
| **クリニック** | 診療機能トグル |
| **ビジネス** | 組織情報 |
| **スタッフ** | チーム権限 |
| **空き状況** | デフォルトスケジュール |
| **保険** | 保険プロバイダー |
| **コミュニケーション** | 通知設定 |
| **統合** | 外部接続 |

---

## アカウント設定

ログインしているユーザーの個人アカウント設定。

### プロファイル情報

| フィールド | 説明 |
|------------|------|
| 名 | あなたの名 |
| 姓 | あなたの姓 |
| メール | アカウントメール（ログイン） |
| 電話 | 連絡先番号 |

### パスワード変更

1. 現在のパスワードを入力
2. 新しいパスワードを入力
3. 新しいパスワードを確認
4. **パスワードを更新**をクリック

**パスワード要件：**
- 最低8文字
- 文字と数字の混合を推奨
- 一般的なパスワードは不可

### 二要素認証（近日公開）

- セキュリティ強化のため2FAを有効化
- 認証アプリまたはSMSオプション
- バックアップコードを提供

### 通知設定

受け取るメールを制御：

| 通知 | 説明 |
|------|------|
| 予約リマインダー | 今後の予約のメール |
| 新しいメッセージ | 患者メッセージのアラート |
| 日次サマリー | 毎日の診療サマリー |
| マーケティング更新 | Genkō製品の更新 |

## クリニック設定

機能トグルと診療全体の設定。

### 機能トグル

機能を有効または無効化：

| 機能 | 説明 |
|------|------|
| **遠隔医療** | ビデオ相談を有効化 |
| **患者ポータル** | 患者セルフサービスを許可 |
| **オンライン予約** | 公開予約受付 |
| **SMSリマインダー** | テキストメッセージリマインダー |
| **AIメッセージング** | AI支援患者チャット |

### 請求設定

| 設定 | 説明 |
|------|------|
| デフォルト通貨 | 主要通貨（USD、JPYなど） |
| 税率 | 適用される税率 |
| 請求書プレフィックス | カスタム請求書番号付け |

### 予約設定

| 設定 | 説明 |
|------|------|
| デフォルト所要時間 | 標準予約長さ |
| バッファ時間 | 予約間の分数 |
| 予約ウィンドウ | 患者がどれだけ先まで予約できるか |
| キャンセルポリシー | 必要な通知時間 |

## スタッフ設定

チーム管理と権限。

### 役割ベースのアクセス制御（RBAC）

各役割ができることを設定：

#### 管理者権限
- すべての機能への完全なアクセス
- 変更不可

#### プロバイダー権限

| 権限 | デフォルト |
|------|----------|
| すべての患者を表示 | ✅ |
| 患者記録を編集 | ✅ |
| 予約を管理 | ✅ |
| 遠隔医療アクセス | ✅ |
| 分析を表示 | ✅ |
| スタッフを管理 | ❌ |

#### 看護師権限

| 権限 | デフォルト |
|------|----------|
| すべての患者を表示 | ✅ |
| 基本患者情報を編集 | ✅ |
| 医療記録を表示 | ✅ |
| 医療記録を編集 | ❌ |
| 予約を管理 | ✅ |
| 遠隔医療アクセス | ✅ |

#### スタッフ権限

| 権限 | デフォルト |
|------|----------|
| 基本患者情報を表示 | ✅ |
| 患者記録を編集 | ❌ |
| 医療記録を表示 | ❌ |
| 予約を管理 | ✅ |
| 遠隔医療アクセス | ❌ |

### 権限のカスタマイズ

1. 役割セクションを見つける
2. 個々の権限をオン/オフにトグル
3. **変更を保存**をクリック

---

## 空き状況設定

デフォルトスケジュール設定。

### 営業時間

デフォルトの営業時間を設定（公開予約に影響）：

| 設定 | 説明 |
|------|------|
| 営業日 | 営業する曜日 |
| 開始時間 | 毎日の開店時間 |
| 終了時間 | 毎日の閉店時間 |
| タイムゾーン | 営業タイムゾーン |

### 予約タイプ

利用可能な予約タイプを設定：

| タイプ | 所要時間 | 色 |
|--------|----------|-----|
| 新規患者 | 60分 | 青 |
| フォローアップ | 30分 | 緑 |
| 年次検診 | 45分 | 紫 |
| 相談 | 20分 | オレンジ |
| 遠隔医療 | 30分 | シアン |

### 予約タイプの追加

1. **タイプを追加**をクリック
2. 名前と所要時間を入力
3. 表示色を選択
4. 空き状況制限を設定
5. **保存**をクリック

### 予約ルール

| ルール | 説明 |
|--------|------|
| 最低通知 | 予約可能になるまでの時間 |
| 最大先 | 予約可能な先の日数 |
| 当日予約 | 当日を許可/禁止 |
| 重複防止 | ダブルブッキングをブロック |

## 保険設定

受け入れ保険プロバイダーを管理。

### 保険プロバイダーリスト

設定された保険会社のテーブル：

| 列 | 説明 |
|-----|------|
| プロバイダー名 | 保険会社名 |
| 支払者ID | 電子支払者ID |
| ステータス | アクティブ/非アクティブ |
| アクション | 編集、削除 |

### 保険プロバイダーの追加

1. **プロバイダーを追加**をクリック
2. 詳細を入力：
   - プロバイダー名
   - 支払者ID
   - 連絡先電話
   - 送信タイプ（電子/紙）
3. **保存**をクリック

### 保険プロバイダーの編集

1. プロバイダー行の編集アイコンをクリック
2. フィールドを修正
3. **保存**をクリック

### プロバイダーの無効化

1. ステータスを非アクティブにトグル
2. プロバイダーは新しい請求から非表示
3. 既存の請求は保存

## 設定の保存

### 自動保存

一部の設定は自動的に保存：
- トグルスイッチ
- ドロップダウン選択
- 個々のフィールド更新

### 手動保存

その他は明示的な保存が必要：
- 複雑なフォーム
- 一括変更
- 権限マトリックス

**確認：**
- 「保存」確認トースト
- 「保存」ボタン状態の変化
- 保存中のローディングスピナー

## ベストプラクティス

### 初期設定

1. 最初にビジネス設定を完了
2. 保険プロバイダーを設定
3. 空き状況のデフォルトを設定
4. 予約タイプをカスタマイズ
5. スタッフ権限を確認

### 定期メンテナンス

1. 四半期ごとに権限を確認
2. 年次で保険を更新
3. 連絡先情報を確認
4. 統合を定期的にテスト

### セキュリティ

1. 強力なパスワードを使用
2. 利用可能になったら2FAを有効化
3. アクセスログを確認
4. 権限の付与を最小限に

---

## トラブルシューティング

### 設定が保存されない

- インターネット接続を確認
- 検証エラーを探す
- エラーメッセージを見つけるためにスクロールダウン
- 更新して再入力を試す

### 権限変更が機能しない

- ユーザーはログアウトして再ログインが必要かもしれない
- ブラウザキャッシュをクリア
- 正しい役割が選択されているか確認

### 統合接続が失敗

- 資格情報が正しいか確認
- サードパーティサービスが利用可能か確認
- エラーメッセージ詳細を確認
- サポートに連絡
