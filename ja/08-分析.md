# 分析ダッシュボード

分析ダッシュボードは、クリニックのパフォーマンスに関する包括的な洞察を提供します。収益、請求、支払い、主要業績評価指標（KPI）を追跡します。

---

## 分析へのアクセス

1. サイドバーの**分析**をクリック
2. URL：`/admin/analytics`

---

## ダッシュボード概要

分析ページには以下が含まれます：
- **ヘッダー**：タイトルとアクションボタン
- **フィルター**：日付範囲とレポートタイプ
- **KPIカード**：一目でわかる主要メトリクス
- **チャート**：ビジュアルデータ表現
- **詳細レポート**：タブ付き分析セクション

---

## フィルターとコントロール

### 日付範囲選択

分析期間を設定：

| フィールド | 説明 |
|------------|------|
| 開始日 | 分析期間の開始 |
| 終了日 | 分析期間の終了 |

**クイック日付範囲：**
- 過去7日
- 過去30日
- 過去90日
- 今月
- 今四半期
- カスタム範囲

### レポートタイプ

分析タイプを選択：

| レポートタイプ | 表示 |
|---------------|------|
| **フルダッシュボード** | すべてのメトリクスとチャート |
| **収益サマリー** | 収益中心のデータ |
| **請求分析** | 保険請求詳細 |
| **支払い分析** | 支払い追跡 |

### 適用と更新

- **フィルターを適用**をクリックしてデータを更新
- **更新**（🔄）をクリックして現在のビューをリロード
- **エクスポート**（📥）をクリックしてレポートをダウンロード

---

## KPIカード

4つの主要メトリクスカードが上部に表示：

### 1. 総収益
- **値**：選択期間の総収益
- **アイコン**：💵 ドル記号
- **トレンド**：上昇/下降インジケーター
- **色**：緑

### 2. 請求総数
- **値**：送信された請求数
- **アイコン**：📄 ドキュメント
- **トレンド**：ニュートラルインジケーター
- **色**：青

### 3. 請求支払率
- **値**：支払われた請求の割合
- **アイコン**：✅ チェックマーク
- **トレンド**：上昇（>80%）または下降
- **色**：緑または赤

### 4. 保留中の請求
- **値**：処理待ちの請求
- **アイコン**：⏰ 時計
- **トレンド**：ニュートラル
- **色**：黄

---

## 分析タブ

### 概要タブ

以下を表示するデフォルトビュー：

**収益チャート：**
- 月次収益トレンド
- 折れ線または棒グラフの視覚化
- 前期間との比較

**請求分析：**
- ステータス別請求（円グラフ）
- 承認、拒否、保留中の内訳
- 処理時間の平均

**トップ支払者：**
- 保険支払者のランキングリスト
- 支払い金額
- 総収益の割合

---

### 収益タブ

詳細な収益分析：

**収益トレンドチャート：**
- 月次/四半期別内訳
- 時間経過による収益
- 目標との比較

**ソース別収益：**
- 保険支払い
- 患者支払い
- その他の収入

**収益メトリクス：**
| メトリクス | 説明 |
|------------|------|
| 総収益 | 総収益 |
| 訪問あたり平均 | 予約あたりの収益 |
| 回収率 | 請求額の回収割合 |

---

### 請求タブ

保険請求分析：

**請求ステータス概要：**
| ステータス | 説明 |
|------------|------|
| 承認 | 保険で支払われた請求 |
| 拒否 | 却下された請求 |
| 保留中 | 審査中 |
| 異議申立 | 拒否に対して争っている |

**請求分析チャート：**
- ステータス分布（円グラフ）
- 支払者別請求（棒グラフ）
- 拒否率トレンド

**請求メトリクス：**
| メトリクス | 説明 |
|------------|------|
| 請求総数 | 請求数 |
| 承認率 | 承認された割合 |
| 平均処理時間 | 解決までの日数 |
| 拒否率 | 拒否された割合 |

---

### 支払いタブ

支払い追跡と分析：

**支払いソース：**
| ソース | 説明 |
|--------|------|
| 保険 | 第三者支払者 |
| 自己負担 | 患者直接支払い |
| 現金 | オフィス内現金 |
| カード | クレジット/デビットカード |

**支払い方法：**
- 現金
- クレジットカード
- デビットカード
- 銀行振込
- 小切手

**最近の支払い：**
- 最新の支払い取引リスト
- 金額、ソース、方法、ステータス
- 支払いアクティビティのクイックビュー

**支払いメトリクス：**
| メトリクス | 説明 |
|------------|------|
| 支払い総額 | すべての支払いの合計 |
| 支払い件数 | 取引数 |
| 平均支払い | 平均支払い金額 |

---

## 主要業績評価指標（KPI）

### KPIセクション

診療所用に設定されたKPIを表示：

| KPI | 説明 | 目標 |
|-----|------|------|
| 回収率 | 請求額の回収% | 95% |
| 拒否率 | 請求拒否% | <5% |
| 売掛金日数 | 回収までの平均日数 | <30 |
| 患者数 | 月間患者数 | 変動 |
| 訪問あたり収益 | 予約あたりの平均収益 | 変動 |

### KPIカード表示

各KPIには以下が表示：
- KPI名
- カテゴリバッジ
- 説明
- 目標値（設定されている場合）
- 現在値（計算時）
- アラートステータス

## チャートと視覚化

### 使用されているチャートタイプ

| チャートタイプ | 用途 |
|---------------|------|
| 折れ線グラフ | 時間経過によるトレンド |
| 棒グラフ | カテゴリ比較 |
| 円グラフ | 分布の内訳 |
| エリアチャート | 時間経過による量 |

### チャートの操作

- **ホバー**：データポイント詳細を表示
- **凡例クリック**：データシリーズのトグル
- **ズーム**：スクロールして日付範囲を調整
- **エクスポート**：チャート画像をダウンロード

## データのエクスポート

### エクスポートオプション

1. **エクスポート**ボタンをクリック
2. フォーマットを選択：
   - PDFレポート
   - Excelスプレッドシート
   - CSVデータ

### 含まれる内容

| レポートタイプ | 内容 |
|---------------|------|
| フルダッシュボード | すべてのメトリクスとチャート |
| 収益レポート | 収益データとトレンド |
| 請求レポート | 請求分析データ |
| 支払いレポート | 支払い取引 |

---

## データの更新

### 自動更新

- フィルターを変更するとデータが更新
- 重要なメトリクスのリアルタイム更新
- キャッシュされたデータは最大15分古い可能性

### 手動更新

1. **更新**ボタン（🔄）をクリック
2. データのリロードを待つ
3. 「更新中」スピナーは更新進行中を示す

**レポート情報：**
- 生成タイムスタンプ

- レポートタイプ

## メトリクスの理解

### 収益健全性インジケーター

| メトリクス | 良好 | 警告 | アクション必要 |
|------------|------|------|--------------|
| 回収率 | >95% | 85-95% | <85% |
| 拒否率 | <5% | 5-10% | >10% |
| 売掛金日数 | <30 | 30-45 | >45 |

### メトリクスの改善

**低い回収率：**
- 請求手順を見直す
- 未払い残高をフォローアップ
- 事前に保険情報を確認

**高い拒否率：**
- 一般的な拒否理由を監査
- 適切なコーディングでスタッフを訓練
- サービス前に資格を確認

**高い売掛金日数：**
- フォローアップ手順を改善
- 迅速に請求を送信
- 拒否に迅速に異議申立

## ベストプラクティス

### 定期的なレビュー

1. 緊急の問題について毎日ダッシュボードを確認
2. トレンドについて毎週レビュー
3. 戦略的計画のために毎月深堀り
4. 季節パターンについて四半期ごとに比較

### フィルターの効果的な使用

1. 概要のために広い日付範囲から開始
2. 問題を調査するために絞り込む
3. 類似期間を比較（月対月）
4. 特定の支払者またはサービスを追跡

### データに基づくアクション

1. 主要メトリクスの目標を設定
2. 進捗を定期的にモニター
3. 異常を直ちに調査
4. チームと洞察を共有

---

## トラブルシューティング

### データが読み込まれない

- インターネット接続を確認
- 日付範囲が有効か確認
- より狭い日付範囲を試す
- ページを更新

### 不正確なデータ

- フィルター設定を確認
- データ入力の正確性を確認
- 同期を待つ（最大15分）
- 不一致についてサポートに連絡

### チャートが表示されない

- 別のブラウザを試す
- JavaScriptを有効化
- ブラウザキャッシュをクリア
- 広告ブロッカーを確認

### エクスポートが機能しない

- ブラウザのダウンロード設定を確認
- サイトでポップアップを許可
- 別のフォーマットを試す
- 日付範囲サイズを縮小
