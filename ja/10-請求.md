# 請求とサブスクリプション

請求セクションでは、アカウントオーナーがサブスクリプション、支払い方法、請求書を管理できます。アカウントオーナーのみが請求機能にアクセスできます。

---

## 請求へのアクセス

1. サイドバーの**請求**をクリック
2. URL：`/admin/billing`

**注意：** アカウントオーナーのみが請求セクションにアクセスできます。サイドバーに表示されない場合は、組織のオーナーに連絡してください。

---

## 請求概要

請求ページには以下が表示されます：
- **現在のプラン**：アクティブなサブスクリプション
- **使用統計**：制限に対する機能使用状況
- **支払い方法**：保存された支払い情報
- **請求履歴**：過去の請求書と取引

---

## 現在のサブスクリプション

### プラン情報

| 要素 | 説明 |
|------|------|
| プラン名 | 現在のプラン（スターター、プロフェッショナルなど） |
| ステータス | アクティブ、トライアル、延滞、キャンセル済み |
| 請求サイクル | 月次または年次 |
| 価格 | 月額/年額費用 |
| 更新日 | 次回請求日 |

### プランステータスインジケーター

| ステータス | バッジ色 | 意味 |
|------------|---------|------|
| アクティブ | 🟢 緑 | サブスクリプションは最新 |
| トライアル | 🔵 青 | トライアル期間中 |
| 延滞 | 🟡 黄 | 支払い問題 |
| キャンセル済み | 🔴 赤 | サブスクリプション終了 |

|------|------|----------|------------------|---------|
| 月額価格 | $0 | $29 | $99 | $249 |
| スタッフメンバー | 1 | 3 | 10 | 無制限 |
| 患者 | 10 | 無制限 | 無制限 | 無制限 |
| 予約 | 10/月 | 無制限 | 無制限 | 無制限 |
| 遠隔医療 | ❌ | 5/月 | 無制限 | 無制限 |


## プランの変更

### アップグレード

1. **プランをアップグレード**または**プランを変更**をクリック
2. 利用可能なプランを確認
3. 希望のプランを選択
4. 変更を確認
5. 日割り計算の料金が即時適用

**アップグレード時：**
- 新機能が即座に利用可能
- 残りの請求期間の日割り請求
- より高い制限が即時適用

### ダウングレード

1. **プランを変更**をクリック
2. 下位ティアプランを選択
3. 影響を確認（失う機能）
4. ダウングレードを確認
5. 次回請求サイクルで有効

**ダウングレード時：**
- 現在の機能は期間終了まで利用可能
- 新しい制限は次回請求サイクルで適用
- 未使用時間の返金なし
- 新しい制限を超えるデータは対処が必要

### 請求サイクルの切り替え

月次と年次請求を切り替え：

| サイクル | メリット |
|----------|---------|
| 月次 | 柔軟性、使用に応じて支払い |
| 年次 | 約20%割引、価格固定 |

1. **請求サイクルを変更**をクリック
2. 新しい価格を確認
3. 変更を確認
4. 新しいサイクルは次回更新時に適用

---

## トライアル期間

### トライアル中

- 選択したプラン機能への完全アクセス
- 14日間
- 開始に支払い不要
- トライアル終了前に支払い方法が必要

### トライアル終了が近づいたら

リマインダーを受け取ります：
- トライアル終了7日前
- トライアル終了3日前
- トライアル終了1日前

### トライアル終了後

**支払い方法が追加されている場合：**
- 有料サブスクリプションへの自動変換
- 最初の請求を処理
- サービスの中断なし

**支払い方法がない場合：**
- 無料プラン機能に制限
- データは30日間保存
- いつでもアップグレードしてアクセスを復元

---

## 支払い方法

### 支払い方法の追加

1. **支払い方法を追加**をクリック
2. カード詳細を入力：
   - カード番号
   - 有効期限
   - CVC
   - 請求先住所
3. **保存**をクリック

**サポートされる支払い方法：**
- クレジットカード（Visa、Mastercard、Amex、Discover）
- デビットカード
- 一部の国際カード

### 支払い方法の管理

**デフォルトとして設定：**
- カードの**デフォルトにする**をクリック
- デフォルトカードがすべての請求に使用

**支払い方法の削除：**
- **削除**をクリック
- 有料プランでは少なくとも1つの方法が必要
- 保留中の請求がある場合は削除不可

### 支払い情報の更新

1. 支払い方法の**編集**をクリック
2. 請求先住所を更新
3. カード更新の場合は、古いものを削除して新しいものを追加

## 請求書と領収書

### 請求書の表示

すべての過去の請求書がリストされます：

| 列 | 説明 |
|-----|------|
| 日付 | 請求書日付 |
| 請求書# | 一意の請求書番号 |
| 金額 | 請求総額 |
| ステータス | 支払済み、保留中、失敗 |
| アクション | 表示、ダウンロード |

### 請求書のダウンロード

1. リストで請求書を見つける
2. **ダウンロード**（📥）をクリック
3. PDFが自動的にダウンロード

### 請求書詳細

請求書をクリックして表示：
- 明細項目
- 適用された税金
- 使用された支払い方法
- 取引ID

プラン制限に対する使用状況を追跡：

**スタッフメンバー：**
```
現在：5

ステータス：✅ 制限内
```

**月間予約：**
```
現在：142
制限：無制限
ステータス：✅ 制限なし
```

**遠隔医療セッション：**
```
現在：23
制限：30
ステータス：⚠️ 制限に近づいている
```

### 使用状況アラート

以下の場合にアラートを受け取ります：
- 制限の80%に近づいた場合
- 制限の100%に達した場合
- 制限を超えた場合

## 制限違反

### 制限を超えた場合

プラン制限を超えた場合：

1. **警告バナー**：影響を受けるページに表示
2. **機能制限**：一部の機能が制限される可能性
3. **猶予期間**：問題に対処するための短い期間
4. **解決オプション**：
   - プランをアップグレード
   - 使用を減らす
   - サポートに連絡

### 違反の解決

**オプション1：プランをアップグレード**
- 警告バナーで**アップグレード**をクリック
- より高い制限のプランを選択
- 制限が即座に解除

**オプション2：使用を減らす**
- 未使用のスタッフアカウントを無効化
- 古いデータをアーカイブ
- 次の請求サイクルを待つ

## キャンセル

### サブスクリプションのキャンセル

1. **サブスクリプションをキャンセル**をクリック
2. キャンセル理由を選択
3. 失うものを確認
4. キャンセルを確認

### キャンセル後

- 請求期間終了までアクセス継続
- 追加料金なし
- データは30日間保存
- いつでも再有効化可能

### 再有効化

1. 請求ページに移動
2. **サブスクリプションを再有効化**をクリック
3. プランを選択
4. 支払い方法を追加/確認
5. サブスクリプション復元

---

## 請求サポート

### よくある問題

**支払い失敗：**
1. カード有効期限を確認
2. 請求先住所を確認
3. 銀行に連絡
4. 別のカードを試す

**予期しない請求：**
1. 請求書詳細を確認
2. 日割り請求を確認
3. 不明な場合はサポートに連絡

**返金リクエスト：**
- 30日以内にサポートに連絡
- 請求書番号を提供
- 状況を説明

### 請求サポートへの連絡

1. 請求ページで**サポートに連絡**をクリック
2. またはbilling@genko.appにメール
3. 組織名と請求書番号を含める

---

## ベストプラクティス

### コスト管理

1. 節約のために年次請求を選択
2. 実際のニーズに合わせてプランをサイズ調整
3. 未使用のスタッフアカウントを無効化
4. 使用状況を定期的にモニター

### 支払いセキュリティ

1. 支払い方法を最新に保つ
2. 有効期限前に更新
3. 請求書メールをモニター
4. 疑わしい請求は直ちに報告

### 成長の計画

1. 使用傾向を追跡
2. 制限に達する前にアップグレード
3. 成長するチームの予算を計画
4. 安定性のために年次プランを検討

---

## トラブルシューティング

### 請求にアクセスできない

- アカウントオーナーのみアクセス可能
- 組織のオーナーに連絡
- アカウントの役割を確認

### 支払いが処理されない

- カード詳細を確認
- 十分な資金があるか確認
- 別のカードを試す
- 銀行に連絡

### 請求書が届かない

- スパムフォルダを確認
- 設定でメールアドレスを確認
- 請求ページからダウンロード

### プラン変更が適用されない

- 有効日を確認
- 変更は次サイクルで適用される場合
- 緊急の場合はサポートに連絡

---

## セキュリティとコンプライアンス

### 支払いセキュリティ

- すべての支払いはStripe経由で処理
- PCI-DSS準拠
- カード番号は当社サーバーに保存されない
- 暗号化された送信

### データ保護

- 請求情報はプライベート
- アカウントオーナーのみが請求にアクセス
- 監査ログを維持
- HIPAAコンプライアンスのデータ取り扱い
