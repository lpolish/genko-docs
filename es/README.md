## Cómo Usar Este Manual

Este manual está organizado en el orden en que típicamente configurarías y usarías la plataforma Genkō:

1. **Usuarios nuevos** deben comenzar con [Primeros Pasos](./01-primeros-pasos.md) y trabajar a través de las secciones de configuración.
2. **Operaciones diarias** se cubren en las secciones de Gestión de Pacientes, Programación y Comunicaciones.
3. **Tareas administrativas** se cubren en Gestión de Personal, Análisis y Configuración.

Cada sección incluye:
- Instrucciones paso a paso con marcadores para capturas de pantalla
- Mejores prácticas y consejos
- Problemas comunes y soluciones

---

## Consejos de Navegación

- Usa la navegación del sidebar en el portal de administración para acceder a diferentes secciones
- La barra de búsqueda global (Ctrl/Cmd + K) permite acceso rápido a pacientes, citas y configuración
- Las migas de pan en la parte superior de cada página muestran tu ubicación actual

## Obtener Ayuda

Si necesitas asistencia adicional:
1. Consulta la sección relevante de este manual
2. Usa los iconos de ayuda (?) dentro de la aplicación para orientación contextual
3. Contacta soporte a través de la sección de Soporte de la aplicación
